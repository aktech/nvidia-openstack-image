======================================================
NVIDIA CUDA Drivers Installation for CentOS 8
======================================================

This element will download and install NVIDIA CUDA Drivers compiled for kernel release used in guest image


Requires:
- CUDA repository installed on build host
- CUDA device on build host
- export DIB_YUM_REPO_CONF=<cuda repo location on build host> 
- export CUDA_DOWNLOAD_PATH=<cuda run file installter url>
